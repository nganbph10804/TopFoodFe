# TopFood-FE

Đặt tên branch theo chức năng:

```
họ tên-tên function
eg: tdm-login
```

Đặt tên commit theo format:

```
[họ tên] "tên chức năng" "hậu tố"
note: hậu tố là phần fix bug commit lại branch thì mới fill.
VD: [TDM] login -fix-done
```
